Задача:

Реализовать алгоритм для поиска оптимального пути во взвешенном неориентированном графе.

Входные данные:
    
    Файл формата:
        [v1 w1 l1]
        ...
        [vn wn ln]        
        Каждая столка задает ненаправленное ребро (v,w) с весом l        
    Стартовая вершина    
    Конечная вершина

Выходные данные:
    
    Путь в графе    
    Сумма 

В программе реализован алгоритм Дейкстры. Сложность O(n^2)
